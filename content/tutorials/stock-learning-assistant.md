# 我让 Claude 帮我建了个私人美股学习助手

> 本教程配套小红书视频，[点击查看视频](http://xhslink.com/o/9nMzLSrMx4B)。

## 目的

想快速搭建对美股的认知体系，但 YouTube 上资源太多无从下手。于是用 Claude 的 Cowork 模式，2 小时完成了原本需要 2 天的工作——从 YouTube 筛选 50 个高质量美股学习视频，按主题分类整理，并全部导入 NotebookLM 打造私人 AI 学习助手。

## 提示词

```
假如你是一个美股专家，有近20年美股炒股经历，并且胜率和回报率都能超过全世界99%的投资者，同时你也是个美股教育专家，可以教会初学者从0-1学习美股并实操。

我是一个有金融背景，但是没有美股炒股经验的小白，需要非常快速搭建对美股的认知，比如快速了解美股细分行业（比如高关注度赛道，以及都有什么比较有潜力的赛道）、个股行情（比如什么股票2026年有机会，什么股票现在是高位，什么股票是低估且有潜力在今年上涨的，什么公司2026年将ipo等等）、如何交易美股、在基本面和技术层面需要学习并注意什么。同时，也得实操导向的，能让我快速上手炒股，推荐多种特质的标的，比如高回报率高风险、稳健收益的等等。

首先，请你从youtube认真筛选50个非常高质量、有信息量、符合上述要求的视频，时效性方面你自己评估，比如和行情实效关系大的信息当然是最新视频比较好。

其次，把这些视频导入notebooklm方便我之后学习
```

## 效果

Claude 自动完成了以下步骤：

1. 联网搜索各主题的优质频道和视频
2. 按入门、进阶、实战逻辑分类整理
3. 将所有视频链接整理成文档
4. 自动打开浏览器，逐个添加到 YouTube 播放列表
5. 全部导入 NotebookLM，变成私人 AI 学习助手

现在 NotebookLM 里有了专属美股学习库，想复习什么直接问 AI，它会基于这 50 个视频的内容回答。
