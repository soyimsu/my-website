# 用 NotebookLM + 播客秒变私人英语外教

> 本教程配套小红书视频，[点击查看视频](http://xhslink.com/o/7fOsigehwHC)。

## 目的

把一期感兴趣的英文播客丢进 NotebookLM，用 7 个提示词系统性地拆解语料——提取高级词汇、积累地道表达、练习口语复述和对话模拟，最后还能出题检测学习成果。一期播客就能收获十几个高级词汇、十几个地道表达和现成的对话素材。

## 示例播客

Hidden Brain（心理学主题英文播客）

## 提示词

### 1. 词汇提取

```
请从这期播客中提取 15 个中高级词汇或短语，要求：

1. 这些词我可能认识，但不太会主动使用
2. 每个词给出：
   - 英文释义（用简单英语解释）
   - 中文翻译
   - 播客中的原句
   - 一个新的例句，展示如何在日常对话中使用
3. 标注这个词适合用在口语还是书面语，或者两者都可以
```

### 2. 积累口语表达

```
从这期播客中找出 10 个地道的口语表达，这些表达要能替代中国学生常用的「作文腔」说法。

比如：
- 替代 "I think" 的更自然说法
- 替代 "very important" 的更地道表达
- 表示同意、不同意、转折的自然说法

每个表达请给出：
1. 播客中的原句
2. 为什么这个表达比教科书说法更自然
3. 2 个可以直接套用的场景例句
```

### 3. 分析对话技巧

```
分析这期播客中，主持人和嘉宾是如何做到以下几点的：

1. 开始一个新话题（找 2 个例子）
2. 表示部分同意但有保留（找 2 个例子）
3. 举例说明一个观点（找 2 个例子）
4. 总结或收尾（找 2 个例子）
5. 在思考时使用的填充词或过渡语（找 3 个例子）

每个例子请：
- 引用原文
- 解释这个表达的功能
- 给出一个我可以模仿使用的场景
```

### 4. 口语复述练习

```
请帮我准备一个 2 分钟的口语复述，内容是向朋友介绍这期播客的主要观点。

要求：
1. 用口语化的英语，不要太书面
2. 包含这期播客的 3 个核心观点
3. 用到至少 5 个我们之前提取的地道表达
4. 在适当的地方标注 [停顿] 和 [重读]，帮助我练习语调
5. 控制在 250-300 词左右
```

### 5. 对话模拟

```
基于这期播客的内容，生成一段两人对话。

场景：两个朋友在咖啡厅聊天，A 最近感觉工作陷入瓶颈，B 听了这期播客后分享了一些想法。

要求：
1. 对话要自然、口语化，像真实朋友聊天
2. 包含至少 8 个我们之前学过的表达
3. 对话长度：每人 6-8 轮发言
4. 在对话后，用【】标注出哪些是值得学习的表达
5. A 和 B 的观点可以有分歧，体现真实讨论的感觉
```

### 6. 讨论问答

```
基于这期播客的内容，请扮演一个英语口语教练，用英语问我 5 个讨论问题。

要求：
1. 问题要能引发思考，不是简单的 yes/no 问题
2. 每个问题后给出 2-3 个可以用到的关键表达，帮助我回答
3. 问题难度递进，从简单到需要更多思考
```

### 7. 一周后复习测验

```
请根据这周我们学习的所有内容，给我出一个小测验：

1. 5 道词汇填空题（用我们学过的词）
2. 3 道表达改写题（把「作文腔」改成地道表达）
3. 1 道口语情景题（给我一个场景，让我用学过的表达来应对）

每道题给出答案和解析。
```

## 效果

7 个提示词用完，一期播客就能收获：

- 15 个中高级词汇（带例句和使用场景）
- 10 个替代「作文腔」的地道口语表达
- 对话技巧分析和现成的对话素材练口语
- 复述、模拟对话、讨论问答等多维度练习
- 一周后的复习测验巩固学习成果

内容来自自己感兴趣的播客，学起来完全不枯燥。
