# 5 个让工作效率翻倍的 AI 自动化工作流

> 本教程配套小红书视频，以下是视频中提到的工具和 Prompt。

## 工作流 1：自动整理会议纪要

**工具组合**：录音软件 + Claude

录音转文字后，使用以下 Prompt 整理：

```
请将以下会议录音转写内容整理成结构化会议纪要，包含：
1. 会议主题
2. 参会人员
3. 讨论要点（分条列出）
4. 待办事项（标注负责人和截止日期）
5. 下次会议安排

转写内容：
[粘贴内容]
```

## 工作流 2：批量生成社交媒体文案

**工具组合**：Claude + 批量处理

```
你是一位资深的社交媒体运营专家。请根据以下主题，生成 5 条小红书风格的文案：
- 主题：[你的主题]
- 风格：口语化、有干货、带 emoji
- 每条文案 150-200 字
- 每条附带 5 个相关标签
```

## 工作流 3：自动化数据报告

**工具组合**：Excel/CSV + Claude

```
请分析以下数据并生成一份简报：
1. 数据概览（总量、均值、趋势）
2. 关键发现（Top 3）
3. 异常值分析
4. 建议下一步行动

数据：
[粘贴数据]
```

## 工作流 4：邮件自动分类与回复

**工具组合**：Claude

```
请帮我分析以下邮件，并：
1. 判断优先级（高/中/低）
2. 归类（需回复/仅知悉/可忽略）
3. 对需要回复的邮件，草拟回复内容

邮件内容：
[粘贴邮件]
```

## 工作流 5：代码审查与优化

**工具组合**：Claude Code

直接在 VSCode 终端中：

```bash
claude "请审查当前文件的代码，指出潜在的 bug、性能问题和改进建议"
```
