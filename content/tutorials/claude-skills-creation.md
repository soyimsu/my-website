# 2 句话让 AI 帮你写一个专属 Skill

> 本教程配套小红书视频，[点击查看视频](http://xhslink.com/o/2VlPZCHVih0)。

## 目的

把你经常重复做的工作变成一个 Skill（AI 的专属 SOP），以后只需要一句触发词就能自动执行，输出质量稳定、格式统一。本文以"自动生成美股行情日报"为例，演示如何用 2 步创建自己的 Skill。

## 什么是 Skill

Skill 就是给 AI 写的一份超详细工作手册，和普通 Prompt 的区别：

- **普通 Prompt**：临时交代任务，每次都要重新说要求，AI 可能忘记标准
- **Skill**：提前写好的 SOP，包含工作流程、格式要求、输出标准，一句触发词就能执行

适用场景：自动生成日报/周报、批量处理 Excel 数据、固定格式写邮件文档、自动化工作流——只要是经常重复做的事，都可以变成 Skill。

## 创建步骤

### 第 1 步：安装 Skill Creator

Skill Creator 是"用来写 Skill 的 Skill"，相当于 Skill 之母。直接对 Claude 说：

```
请帮我安装skill creator，项目地址是 https://github.com/anthropics/skills/tree/main/skills/skill-creator
```

### 第 2 步：描述你的需求

告诉 AI 你想创建什么 Skill，比如：

```
我想创建一个分析美股的Skill，我会给它Finviz网站的截图，帮我生成一份适合新手看的中文分析报告
```

AI 会生成 Skill 初稿，包括触发词、工作流程、输出格式。哪里不满意直接告诉 AI 修改，反复迭代到满意为止。最后 AI 会打包成 .skill 文件，点击添加即可使用。

## 效果

AI 帮我生成了一个 600 多行的专业 SOP，包含：

- **触发条件**：说"Finviz 美股日报"并发送截图时触发
- **工作流程**：自动判断盘中/盘后 → 提取截图数据 → 搜索个股新闻 → 生成报告 → 保存到 Notion
- **输出格式**：11 个板块，每个板块规定了表格样式、字数要求和必备内容

每天只需发两张截图，就能出一份几千字的专业行情报告。用了一段时间发现不好的地方，还能随时让 AI 修改，越用越顺手。
